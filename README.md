# Diyasha Nag | Portfolio Website

A modern, dark-themed responsive portfolio website showcasing my skills, projects, and experience as a Computer Science student and Full-Stack Developer.

## Features

- **Dark Theme**: Modern dark color scheme with vibrant accents
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Glassmorphism Effects**: Modern UI with backdrop blur and transparency
- **Fixed Navigation**: Sticky navigation bar with active section highlighting
- **Mobile Menu**: Hamburger menu for mobile devices
- **Smooth Scrolling**: Enhanced navigation experience with scroll animations
- **Sections Include**:
  - Hero section with gradient background
  - About me - KIIT student with 9.41 CGPA
  - Skills in Web Dev, AI/ML, and Android
  - Education & Work experience timeline
  - Featured projects (MEDI-VAULT, Object Detection, Gesture Recognition)
  - Contact information with social links

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A text editor (VS Code, Sublime Text, etc.)

### Installation

1. Clone or download this repository
2. Navigate to the project directory
3. Open `index.html` in your web browser

### Customization

1. **Personal Information**: Edit `index.html` to add your:
   - Name and title
   - About section content
   - Skills and technologies
   - Work experience
   - Projects
   - Contact information

2. **Styling**: Modify `css/style.css` to customize:
   - Color scheme (CSS variables in `:root`)
   - Fonts
   - Spacing and layout

3. **Images**: Add your project screenshots and profile images to the `images/` folder

4. **Projects**: Update the projects section with your actual work, including:
   - Project titles
   - Descriptions
   - Technologies used
   - Links to live demos and GitHub repositories

## Dark Theme Customization

The dark color scheme can be customized by modifying the CSS variables in `css/style.css`:

```css
:root {
    --primary-color: #60a5fa;      /* Blue accent */
    --accent-color: #fbbf24;       /* Gold highlight */
    --text-color: #e5e7eb;         /* Light text */
    --bg-color: #0f172a;           /* Dark background */
    --bg-secondary: #1e293b;       /* Secondary dark */
    --gradient-start: #1e3a8a;     /* Gradient blue */
    --gradient-end: #7c3aed;       /* Gradient purple */
}
```

## Deployment

### GitHub Pages

1. Push your code to a GitHub repository
2. Go to repository Settings > Pages
3. Select the main branch as the source
4. Your site will be published at `https://yourusername.github.io/repository-name`

### Other Options

- **Netlify**: Drag and drop the project folder
- **Vercel**: Connect your GitHub repository
- **Surge**: Use the Surge CLI for quick deployment
