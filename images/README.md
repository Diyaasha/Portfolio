# Project Images

Add your project screenshots here with the following names:

- `medi-vault.jpg` - Screenshot of MEDI-VAULT project
- `object-detection.jpg` - Screenshot of Object Detection System
- `gesture-recognition.jpg` - Screenshot of Gesture Recognition System

## Recommended Image Specifications

- **Format**: JPG or PNG
- **Dimensions**: 800x600 pixels (4:3 ratio) or higher
- **File size**: Under 500KB for optimal loading

## Tips

- Use high-quality screenshots of your projects
- Ensure images are well-lit and clear
- Consider adding overlays or branding if desired
- Compress images for web performance

For now, the project cards will display gradient backgrounds until you add images.
